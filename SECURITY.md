# Security Policy

## Supported Versions

The latest minor release receives security patches.

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |
| older   | No        |

## Reporting a Vulnerability

Please report vulnerabilities privately via GitHub's **Private Security Advisory** feature on this repository. Do not open public issues for security reports.

- Acknowledgement SLA: 3 business days
- Triage SLA: 10 business days
- Fix target: 30 days for high/critical, 90 days for low/medium

## Scope

**In scope**: the API, web app, desktop app, mobile app, and shared packages in this repository.

**Out of scope**: third-party services, upstream tools (yt-dlp, ffmpeg), user-configured reverse proxies, and social-engineering attacks.

## Credit

Reporters are credited in release notes unless anonymity is requested.
